Storage

Block, File and Object

Block Storage

  • 데이터를 고정된 사이즈의 블록으로 나누어 각각 고유한 식별자와 함게 저장하는 방식

File Storage

  • 데이터는 계층적 파일 디렉터리 내의 폴더에 저장됨

Object Storage

  • Object라고 불리는 개별 유닛에 데이터를 저장하는 스토리지 포맷
  • 각 유닛에는 고유의 식별자 혹은 키가 있어서 분산된 시스템 내 어디에 저장되어 있든지 상관없이 데이터를 찾을 수 있음
