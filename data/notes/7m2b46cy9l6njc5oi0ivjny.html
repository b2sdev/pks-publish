Azure Event Grid

  • 서버리스 이벤트 브로커
  • 이벤트 기반 아키텍처를 사용하여 솔루션을 연결할 수 있음
    • 이벤트를 사용하여 다른 애플리케이션이나 서비스와 같은 시스템 상태 변경 내용을 전달
    • 필터를 사용하여 다른 엔드포인트에 대한 특정 이벤트를 라우팅하고, 여러 엔드포인트로 멀티캐스트하며, 이벤트가 안정적으로 배달될 수 있도록 할 수 있음
  • 이벤트 처리기
    • Azure 함수
    • Event Hubs
    • Service Bus 큐 및 항목
