## OLTP(Online Transactional Processing)
- 온라인 트랜잭션 처리
    - 데이터베이스의 데이터를 수시로 갱신하는 프로세싱
    - 호스트 컴퓨터가 데이터베이스를 액세스하고, 바로 처리 결과를 돌려주는 형태
- 행 기반
- 기존 DBMS
- Amazon RDS

## OLAP(Online Analytical Processing)
- 온라인 분석 처리
    - 대용량 데이터 집합을 대상
- 컬럼 기반
- Amazon Redshift
