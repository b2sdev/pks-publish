## Notebook

* Notebook에서 library 설치
```sh
%pip install /dbfs/path/to/file.whl
```

* Notebook에서 shell 스크립트 실행
```sh
%sh
/dbfs/path/to/file.sh
```

* Notebook에서 Python 스크립트 실행
```sh
%run ./path/to/file.py
```
