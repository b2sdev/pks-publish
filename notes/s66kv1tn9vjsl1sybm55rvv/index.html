소인수분해 후 각 소인수의 지수의 합 구하기

문제

1×2×3×⋅⋅⋅×101 \times 2 \times 3 \times ··· \times 101×2×3×⋅⋅⋅×10을 소인수분해하면 2x×3y×5z×72^x \times 3^y \times 5^z \times 72x×3y×5z×7이다. 이때 자연수 x,y,zx, y, zx,y,z에 대하여 x+y+zx+y+zx+y+z의 값을 구하시오.

풀이

주어진 수를 소인수분해 후 각 소인수의 지수를 모두 합산한다.

from collections import Counter

num = 1
for n in range(2, 11):
    num *= n

factors = Counter(prime_factorization(num))

x = factors[2]
y = factors[3]
z = factors[5]

print(x + y + z)

def prime_factorization(n):
    """Return the prime factorization of a number."""
    factors = []
    divisor = 2  # 시작하는 소수

    while divisor <= n:
        if n % divisor == 0:  # n이 divisor로 나누어 떨어지면
            factors.append(divisor)
            n //= divisor   # n을 divisor로 나눈 몫을 새로운 n으로 설정
        else:
            divisor += 1

    return factors

num = 56
print(prime_factorization(num))  # [2, 2, 2, 7] 출력
